673  """
674  Menu item change.

676  Second step : To avoid the cyclic restore or backup, default menu entry
677  is tested to backup/restore message occurence.

679  otherwise our menu entry is correctly preselected like before.
680

682  @type default_entry: int
683 
684  @param mac: MAC address of client machine
685  @type mac: str
686 
687  @param entry: Menu entry order
688  @type entry: int
689 
690  @return: ACK to confirm a correct reception, otherwise ERROR
691  @rtype: str
692  """
693  if not mac in EntryTracking():
694  return succeed("ACK")
695 
696  actual_entry, marked = EntryTracking().get(mac)
697  if marked and actual_entry == int(default_entry):
698  setDefaultMenuItem = True
699  entry = num
700  else :
701  setDefaultMenuItem = False
